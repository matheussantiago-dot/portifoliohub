# PortfolioHUB + IA Gemini

Protótipo estático desenvolvido em HTML, CSS e JavaScript para a entrega final de implantação do PortfolioHUB.

**Matheus Santiago Valadares Fonseca**  
Estudante do **UniCEUB — Centro Universitário de Brasília**.  
Atualmente aprendendo **linguagem C**, lógica de programação e desenvolvimento web.

## Objetivo

Criar uma página personalizada para apresentar a implantação do PortfolioHUB, integrando:

- GitHub e GitHub Pages;
- documentação do processo;
- boas práticas de segurança;
- apoio da IA Gemini;
- identidade pessoal e acadêmica do estudante.

## Estrutura

- `index.html`: página principal com as seções da implantação.
- `style.css`: layout responsivo e identidade visual personalizada.
- `script.js`: menu mobile, cards dinâmicos, simulador de acesso e checklist de testes.

## Como publicar no GitHub Pages

1. Envie os arquivos para o repositório.
2. Acesse **Settings > Pages**.
3. Selecione a branch `main` e a pasta `/root`.
4. Salve e aguarde a URL pública ser gerada.

## Segurança

- Não publicar tokens, senhas ou chaves de API.
- Usar autenticação em duas etapas no GitHub.
- Controlar permissões por perfil.
- Revisar mudanças antes de publicar.
